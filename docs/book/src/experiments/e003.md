# E-003: Real Cosmological Fit (Pantheon+ / DESI BAO)

**Binary:** `real-cosmo-fit`
**Deterministic:** Yes
**GPU:** No

## Method

Joint chi-square fit over 1578 Pantheon+ Type Ia supernovae and 7 DESI DR1
BAO distance bins.  Analytic M\_B marginalization.  Nelder-Mead optimization.
Compares Lambda-CDM vs w0-CDM via delta-BIC.

## Run command

```sh
cargo run --release --bin real-cosmo-fit
```

## Input

- `data/external/Pantheon+SH0ES.dat` -- Pantheon+ supernova compilation

## Key result

Delta BIC = +7.37 favoring Lambda-CDM over the bounce cosmology model (I-004).
The quantum correction parameter converges to zero, indicating no evidence for
bouncing cosmology in current data.

Critical data quality fix: BGS/QSO bins are isotropic-only (not anisotropic).
