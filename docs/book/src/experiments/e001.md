# E-001: Cayley-Dickson Motif Census

**Binary:** `motif-census`
**Deterministic:** Yes
**GPU:** No

## Method

Exact enumeration of connected-component structure of the diagonal zero-product
graph for cross-assessor pairs at each Cayley-Dickson doubling (dim=16, 32, 64,
128, 256).

## Run command

```sh
cargo run --release --bin motif-census -- --dims 16,32,64,128,256 --details
```

## Output

- `data/csv/motif_census_dim{N}.csv` -- per-dimension component data
- `data/csv/motif_census_summary.csv` -- summary across all dimensions

## Key findings

Five strict scaling laws verified across all 5 doublings (I-006):

- Components = dim/2 - 1
- Nodes per component = dim/2 - 2
- Motif classes = dim/16
- K2 components = 3 + log2(dim)
- K2 part count = dim/4 - 1

No octahedra beyond dim=16; complete structural reorganization at each doubling.

## Claims

C-100 through C-110, C-126 through C-130.
